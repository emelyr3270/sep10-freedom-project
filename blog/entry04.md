<h2> Context </h2>
<p> I was experimenting different tools and learning how to use them for my web design. As I was exploring how to use <a href= "https://wowjs.uk/"> wowjk </a> I noticed that they used animate.css. I remembered working on animation in the AVD jigsaw project with my group. I used my knowledge from that in order to tinker with <a href="https://animate.style/"> animate.css </a> on jsbin. I tried out different animation properties that i didn’t know how to use like animate__zoomInDown. I played around with the other animations they had. I then came across <a href= "https://aframe.io/"> A-frame </a> I tinkered with A-frame responsive UI, i saw that there was something used called <a-entity>. I noticed that they used geometry and positioning. I played around and changed the positioning to see how it affected the object/element. I was also able to change the text by switching its alignment. If I want to change my mind about using animation.css, then ill be using A-frames responsive UI. </p>

<p> A-frame is a web framework for building VR experiences, its based on HTML. It uses entity-component framework. An entity-component is a structural design pattern used primarily in game development to manage game objects. It provides HTML elements that are easy to use called primitives. They modified basic meshes through HTML attributes. </p>

<h3> For example some a-entity components are </h3>

```
<a-box> to <a-entity>
<a-sphere> to <a-entity>
<a-cylinder> to <a-entity>
<a-plane> to <a-entity>
<a-sky> to <a-entity>
```


<h3>These components can be seen in </h3>

```
<html>
  <head>
    <script src="https://aframe.io/releases/1.7.1/aframe.min.js"></script>
  </head>
  <body>
    <a-scene>
      <a-box position="-1 0.5 -3" rotation="0 45 0" color="#4CC3D9"></a-box>
      <a-sphere position="0 1.25 -5" radius="1.25" color="#EF2D5E"></a-sphere>
      <a-cylinder position="1 0.75 -3" radius="0.5" height="1.5" color="#FFC65D"></a-cylinder>
      <a-plane position="0 0 -4" rotation="-90 0 0" width="4" height="4" color="#7BC8A4"></a-plane>
      <a-sky color="#ECECEC"></a-sky>
    </a-scene>
  </body>
</html>

```
